How It Works

System Architecture

A five-stage processing pipeline from raw input to actionable alerts.

Camera / Upload

Webcam stream or uploaded image/video input is ingested into the pipeline.

Image Processing

Frames are resized, normalized, and pre-processed for model inference.

Detection Model

YOLOv8 deep learning model runs object detection on each processed frame.

Classification Engine

Detected regions are classified: Helmet or No Helmet, with confidence scores.

Dashboard & Alerts

Results are displayed on the dashboard and violations trigger instant alerts.
